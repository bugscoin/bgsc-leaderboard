# BGSC Leaderboard

Leaderboard ranking system for BGSC token holders on BNB Chain.

## Features

- Wallet registration
- Real-time ranking by token balance
- Time-weighted average balance calculation
- Multi-language support (EN/KO)
- Binance Wallet integration

## Tech Stack

- React
- Wagmi/Viem
- RainbowKit
- Styled Components
- Framer Motion

## Development

```bash
npm install
npm start
```

## Build

```bash
npm run build
```
